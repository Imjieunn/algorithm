## 내가 푼 방법
⭐️ 방문 처리 및 반복문 ⭐️
- inital[] : 1~n 까지 순서대로 들어있는 숫자 리스트
- flag : 해당 숫자를 제거했는지 여부 파악
- answer[] : 제거한 숫자 순서대로 담는 리스트

        answer 리스트에 모든 숫자가 담길 때까지, 숫자 제거하기
        -> 제거한 숫자를 제외하고 k번째 있는 숫자이며, 해당 숫자를 아직 제거하지 않았다면 -> answer 리스트에 해당 숫자 담고, flag = 1 처리
